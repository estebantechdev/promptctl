Explain the Solution to the Three Boxes Puzzle:

Read the entire description carefully before explaining the answer.

A table has three closed boxes labeled A, B, and C.

Inside the boxes are:

One apple

One book

One coin

Each box contains exactly one item.

The labels on the boxes contain statements:

Box A label: “The apple is not in this box.”

Box B label: “The coin is in Box C.”

Box C label: “The book is in Box A.”

Important rule:
Only one label tells the truth. The other two are false.

Your task is to explain:

Which item is in each box.

Which label is true.

Why the other two labels must be false.

The logical reasoning step by step.
